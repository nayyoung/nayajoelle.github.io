```bash
npm run build
```

This site is automatically deployed to GitHub Pages when changes are pushed to the `main` branch. The GitHub Actions workflow handles building and deploying the site.

### Setup GitHub Pages

1. Go to your repository's Settings > Pages
2. Under "Build and deployment", select "GitHub Actions" as the source
3. The site will automatically deploy on every push to the main branch
